What is the definition of server port?
	- A virtual endpoint on a computer that allows some apps or 	services to send and receive data.
	- Like a "path" to a specific room.

What is server port range?
	- Full set of possible port numbers (0 to 65535)

How many bits does number above represent?
	- 16 bits
